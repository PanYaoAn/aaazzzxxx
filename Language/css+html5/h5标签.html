# h5标签

gt;        表示大于号
%lt;        表示小于号
&nbsp;      表示一个空格
<br />      表示换行
<P> </P>    表示正文段落
<div> </div>      块元素，表示一块内容，没有具体的语义,可以嵌套
<span>      行内元素，表示一行钟的一小段内容，没有具体语义

含样式和语义的标签：
1.<em> </em>表示语气中的强调词，可把文字变成斜体
2.<i> </i>表示专业词汇,可把文字变成斜体
3.<b> </b>行内元素，表示文档钟的关键字和产品名,可把字体加粗
4.<strong></strong>表示非常重要的内容

html图片标签：
<img  />标签：
alt属性，定义图片加载失败时显示的文字，搜索引擎会使用这个文字收录图片
src属性，定义路径

列表标签：
有序列表<ol></ol> 里面嵌套<li> </li>
ol>li*3
无序列表<ul><li> 里面嵌套<li> </li>
ul>li*3

定义列表：dl>(dt+dd)*3
<dl>表示列表整体
<dt>表示列表标题
<dd>表示术语的解释
dt,dd是同层的


<table>标签：
border属性，定义表格的边框，设置值是数值
cellpadding属性，定义单元格与边框的距离，设置值是数值
cellspace属性，定义单元格与单元格之间的距离，设置值是数值
align属性，设置整体表格相对于浏览器窗口的水平对齐方式，设置值为left,center,right

<tr>标签，定义表格中的一行

<td>标签：定义一行中的普通单元格
<th>标签：定义一行中的表头单元格
两个标签的属性：
align设置单元格中内容的水平对齐方式
valign设置单元格中内容的垂直对齐方式，top,middle,bottom
colspan设置单元格水平合并，设置值是数值

rowspan设置单元格垂直合并，设置值是数值
