# CooKie

# 安装

npm install cookie_js --save

# 设置

cookie.set('key', 'value');

# 设置多个值

cookie.set(){
	key1:"value1",
	key2:"value2"
}

如果需要更多选项，如设置过期日期，则可以添加带有选项的对象作为最后一个参数：
cookie.set('key', 'value', {
   expires: 7, // expires in one week
});
 
cookie.set({
   key1: 'value1',
   key2: 'value2'
}, {
   expires: 7
})
key		value
expires	要么是number包含到期日为止的日期，UTCString格式或格式date  object.	浏览器关闭时过期。
domain	A string指定可以访问cookie的域。	当前域。
path	A string这限制了cookie对该路径的访问。	目前的道路。
secure	A boolean指示cookie是否只能通过安全连接访问。	false
sameSite	A string指定基于站点上下文限制cookie访问的SameSite属性。

# 以分钟为单位指定到期日期，则可以配置cookie.expiresMultiplier:

cookie.expiresMultiplier = 1; // Seconds(秒)
cookie.expiresMultiplier = 60; // Minutes.(分钟)
cookie.expiresMultiplier = 60 * 60; // Hours.(小时)
cookie.expiresMultiplier = 60 * 60 * 24; // Days(default).(天)
